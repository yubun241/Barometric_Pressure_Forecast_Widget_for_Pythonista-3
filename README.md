## Barometric_Pressure_Forecast_Widget_for_Pythonista-3
Pythonista 3を利用して、iPhoneのGPSと連動し、現在地の低気圧推移（12時間先まで）をグラフ表示するツールです。
GitHubの `README.md` としてそのまま使える構成案を作成しました。
タイトルのみ英語にし、詳細は日本語で分かりやすくまとめています。


## 概要
このスクリプトは、気象API（Open-Meteo）から取得した気圧データを、iOSネイティブのようなデザインで可視化します。気圧の変化による体調管理（気象病対策）などに役立てることができます。

## 特徴
*   **GPS連動**: アプリ起動時に現在地の緯度・経度を自動取得。
*   **APIキー不要**: 登録不要で使える Open-Meteo API を利用。
*   **12時間予測**: 現在時刻から12時間先までの気圧変化をグラフ化。
*   **ウィジェット対応**: iPhoneの「今日」の表示（ウィジェット）に配置可能。

## 必要な環境
| 項目 | 内容 |
| :--- | :--- |
| **アプリ** | Pythonista 3 (iOS/iPadOS) |
| **ライブラリ** | `requests`, `matplotlib`, `location`, `ui` |
| **権限** | 位置情報の利用許可 |

## 使い方
1.  **コードのコピー**: 下記のソースコードをコピーします。
2.  **Pythonistaで作成**: Pythonista 3を開き、新しいファイル（例: `pressure_graph.py`）を作成してペーストします。
3.  **実行**: 実行ボタンを押すと、位置情報の許可を求められるので「許可」してください。
4.  **ウィジェット登録（任意）**: 
    *   iPhoneのウィジェット画面（ホーム画面の一番左）で「編集」→「カスタマイズ」から Pythonista を追加。
    *   Pythonistaアプリ内の `Settings > Widgets` から、保存したスクリプトを選択。
